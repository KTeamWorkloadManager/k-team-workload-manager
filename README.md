# K Team Workload Manager

## Run locally

1. Install Node.js 18 or newer.
2. In a terminal, open this folder.
3. Run:

```bash
npm install
npm run dev
```

4. Open the local address shown in the terminal, usually `http://localhost:5173`.

## Notes

- Data is currently stored in React state only.
- Refreshing the page resets the sample data.
- Next step is wiring this to a real shared database so your team can use it together.
